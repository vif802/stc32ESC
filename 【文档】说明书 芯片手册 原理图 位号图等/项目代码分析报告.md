## 一、项目概述

这是逐飞科技(SeekFree)开发的**无传感器BLDC(无刷直流)电机控制器**，基于STC32G12K128-LQFP48单片机，专为智能车竞赛设计。采用BEMF(反电动势)过零检测实现无传感器换相控制。

### 主要特点
- **无传感器控制**：无需霍尔传感器，降低电机成本和连线复杂度
- **三阶段平滑启动**：预定位 → 开环加速 → 闭环运行
- **多重安全保护**：低压保护、堵转检测、自动重启
- **S曲线加速算法**：启动过程平滑，减少电流冲击

## 三、核心架构分析

### 3.1 三阶段启动策略

| 阶段 | run_flag | 说明 | 实现 |
|------|----------|------|------|
| **预定位** | 1 | 转子对齐到已知位置 | 固定相通电300ms |
| **开环加速** | 2 | 强制换相加速 | 定时器3递减延时 |
| **过渡阶段** | 3 | PWM中断监控BEMF | 准备切闭环 |
| **闭环运行** | 4 | 比较器中断换相 | 过零检测驱动 |

**代码位置**：`bldc/motor.c:328-400`

### 3.2 中断结构 (按优先级)

| 中断源 | 优先级 | 功能 | 文件位置 |
|--------|--------|------|----------|
| PWMA (PWM) | **高** | BEMF采样(启动阶段) | `motor.c:428` |
| Comparator | **高** | 过零检测(运行阶段) | `motor.c:568` |
| Timer 1 | 普通 | 10ms周期任务(LED/启动逻辑) | `pit_timer.c:88` |
| Timer 3 | 普通 | 开环换相定时 | `motor.c:328` |
| Timer 4 | 普通 | 换相超时检测(堵转) | `motor.c:304` |
| Timer 0 | 普通 | 消磁延时 | `motor.c:410` |
| PWMB | 普通 | PWM输入捕获(油门) | `pwm_input.c:18` |

### 3.3 关键算法实现

#### 3.3.1 加速曲线算法

**代码位置**：`bldc/motor.c:43-69`

支持三种加速曲线：
- **0 - 线性加速**：均匀递减延时
- **1 - 指数加速**：先慢后快
- **2 - S曲线加速(默认)**：使用Sigmoid函数，起步和结尾都较平滑

```c
// S曲线实现
delay = START_OPEN_LOOP_MIN_DELAY +
        (START_OPEN_LOOP_MAX_DELAY - START_OPEN_LOOP_MIN_DELAY) /
        (1.0f + exp((ratio - 0.5f) * 10));
```

#### 3.3.2 堵转检测算法

**代码位置**：`bldc/motor.c:489-502`

通过统计最近6次换相时间的方差判断：
- 最大值/最小值与平均值偏差超阈值
- 或最大值超过15000、最小值小于100

#### 3.3.3 占空比斜坡控制

**代码位置**：`bldc/motor.c:78-94`

开环阶段逐步增加占空比，避免电流冲击：
- 每5ms增加一次占空比
- 每次增加步长为5

## 四、硬件资源分配

### 4.1 PWM输出 (6路中心对齐)

| 相 | 高侧 | 低侧 |
|---|------|------|
| A相 | P20 | P21 |
| B相 | P22 | P23 |
| C相 | P24 | P25 |

### 4.2 BEMF检测

- **比较器正端**：P37(A相) / P51(B相) / P50(C相) - 通过CMPEXCFG寄存器切换
- **比较器负端**：P36(虚拟中性点)

### 4.3 输入信号

- **PWM输入**：P32 (50-300Hz, 1-2ms脉宽)
- **脉宽映射**：1ms(停止) ~ 2ms(满速)

### 4.4 ADC

- **电池检测**：P03 (3S锂电池，低压保护10V)
- **分压比**：57:10 (通过电阻分压网络)

## 七、关键代码流程

### 7.1 电机启动流程

```
motor_start()
├── 1. 初始化与刹车
│   ├── 设置 run_flag = 1 (预定位阶段)
│   ├── pwm_brake() 刹车100ms确保电机静止
│   └── 清除所有状态变量
│
├── 2. 转子预定位 (Alignment)
│   ├── 计算预定位占空比
│   ├── 执行第一次换相
│   └── 配置定时器3，延时后进入开环加速
│
└── 3. 等待定时器3中断进入下一阶段
```

### 7.2 换相控制流程

```
motor_commutation()
├── 根据 motor.step 选择换相方式
│   ├── Step 0: pwm_a_bn_output() + comparator_select_c()
│   ├── Step 1: pwm_a_cn_output() + comparator_select_b()
│   ├── Step 2: pwm_b_cn_output() + comparator_select_a()
│   ├── Step 3: pwm_b_an_output() + comparator_select_c()
│   ├── Step 4: pwm_c_an_output() + comparator_select_b()
│   └── Step 5: pwm_c_bn_output() + comparator_select_a()
│
└── 根据运行阶段设置比较器触发边沿
    ├── 下降沿触发 (falling)
    └── 上升沿触发 (rising)
```

### 7.3 主循环流程

```
main()
├── 系统初始化
│   ├── board_init() - 寄存器初始化
│   ├── battery_init() - 电池检测
│   ├── led_init() - LED初始化
│   ├── pwm_input_init() - PWM输入捕获
│   ├── comparator_init() - 比较器初始化
│   ├── motor_init() - 电机初始化
│   ├── pwm_center_init() - PWM初始化
│   └── pit_timer_init() - 周期定时器
│
└── while(1) 主循环
    └── 发送数据到虚拟示波器 (UART1)
```

## 八、安全保护机制

### 8.1 低压保护

- **检测位置**：`pit_timer.c:92` (10ms周期)
- **阈值**：`BLDC_MIN_BATTERY` (默认10.0V，对应3S锂电3.33V/节)
- **动作**：电压低于阈值时停止电机

### 8.2 堵转检测

- **超时检测**：Timer4监控换相时间，超时则判定堵转
- **统计检测**：通过6次换相时间的方差判断
- **重启机制**：堵转后等待 `BLDC_START_DELAY` (默认0.5秒) 后自动重启

### 8.3 消磁延时

- **Timer0**：换相后延时 `average/4` 时间再检测过零信号
- **目的**：避免换相后的反电动势干扰

## 十、重要注意事项

⚠️ **警告**：PWM初始化必须在 `motor_init()` 之后调用，否则可能烧毁电机！

```c
// 正确顺序
motor_init();           // 先初始化电机相关变量和定时器
pwm_center_init();      // 后初始化PWM
```

⚠️ **注意**：
1. 电机控制使用内部比较器，通过CMPEXCFG寄存器切换三相输入
2. BEMF与虚拟中性点比较，而非GND
3. 启动阶段使用PWM中断采样BEMF，成功后切换至比较器中断

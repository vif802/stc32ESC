# STC32G12K128 无感无刷电调硬件设计说明与采购清单

## 一、系统架构

### 1.1 整体框图

```
3S锂电池(11.1V/12.6V)
    │
    ├──→ 电源管理模块 ──→ SE8650K1-HF ──→ 5V/3.3V
    │                                    │
    └──→ 三相半桥驱动 ──→ FD6288Q ──→ TPN2R203NC ×6 ──→ 无刷电机
                │
                └──→ BEMF分压检测 ──→ STC32G12K128比较器

控制信号(PWM 1-2ms) ──→ STC32G12K128 ──→ FD6288Q
```

### 1.2 功能模块划分

| 模块 | 功能说明 | 核心器件 |
|------|----------|----------|
| 主控单元 | 电机控制算法、PWM生成、BEMF检测 | STC32G12K128-LQFP48 |
| 栅极驱动 | 6路PWM信号放大、高低侧驱动 | FD6288Q |
| 功率开关 | 三相半桥功率MOSFET | TPN2R203NC ×6 |
| 电源管理 | 电池电压转5V/3.3V | SE8650K1-HF + LDO |
| 信号输入 | PWM控制信号接收 | 光耦隔离或直接输入 |
| BEMF检测 | 反电动势分压与虚拟中性点 | 电阻网络 |
| 保护电路 | 过流、欠压、过温保护 | 比较器+采样电阻 |

### 2.3 三相半桥功率电路

#### 2.3.1 拓扑结构

```
VBAT (11.1V-12.6V)
  │
  ├──┬─────────┬─────────┐
  │  │         │         │
  │  │         │         │
 ┌┴──┴┐      ┌┴──┴┐     ┌┴──┴┐
 │AH  │      │BH  │     │CH  │  ← 高侧MOS (TPN2R203NC)
 │    │      │    │     │    │
 └──┬─┘      └──┬─┘     └──┬─┘
    │A         │B        │C         ← 电机相线输出
   ─┴─         ─┴─        ─┴─
   ╱ ╲         ╱ ╲        ╱ ╲        ← 无刷电机
   ╲ ╱         ╲ ╱        ╲ ╱
   ─┬─         ─┬─        ─┬─
    │           │          │
 ┌──┴─┐      ┌──┴─┐     ┌──┴─┐
 │AL  │      │BL  │     │CL  │  ← 低侧MOS (TPN2R203NC)
 │    │      │    │     │    │
 └──┬─┘      └──┬─┘     └──┬─┘
    │           │          │
    └───────────┴──────────┘
                  │
                 GND
```

#### 2.3.2 BEMF检测电路

```
         A相输出 ──→┬─── R1(10k) ──┬───→ P30 (CMP+)
                    │              │
                    │             C5(10nF)
                    │              │
         B相输出 ──→┼─── R2(10k) ──┼───→ P31 (CMP+)
                    │              │
                    │             C6(10nF)
                    │              │
         C相输出 ──→┴─── R3(10k) ──┴───→ P32 (CMP+)
                                   │
                                  R4(1k)
                                   │
                         P33 ──────┴─────┐
                         (CMP-)           │
                                    虚拟中性点
```

**BEMF检测原理：**
- 使用三个10kΩ电阻构成分压网络，将电机相电压分压至MCU可承受范围(0-3.3V)
- 三个10kΩ电阻并联形成虚拟中性点
- 比较器正端接分压后的相电压，负端接虚拟中性点
- 通过检测过零事件判断换相时机

### 2.5 电池电压检测电路

```
VBAT (11.1V-12.6V)
    │
   R5(10k)
    │
    ├──→ P34 (ADC)
    │
   R6(1k)
    │
   GND

分压比：1k/(10k+1k) = 1/11
3S满电12.6V → ADC输入1.145V
3S欠压9V    → ADC输入0.818V
```

### 3.2 关键Layout规则

| 项目 | 要求 | 原因 |
|------|------|------|
| 大电流路径 | 短而粗 | 降低EMI和发热 |
| 栅极驱动线 | 等长，<2cm | 保证死区一致 |
| 功率地与信号地 | 单点连接 | 防止地弹干扰 |
| BEMF检测线 | 远离PWM线 | 避免干扰 |
| 去耦电容 | 紧贴芯片引脚 | 有效滤波 |

## 八、注意事项

1. **MOS管代换**：TPN2R203NC与TPN2R703NL引脚兼容，但耐压/电流不同，确认满足电机功率需求
2. **电源热设计**：SE8650K1-HF在大压差时发热大，建议VBAT<16V使用或加散热
3. **烧录注意**：STC32G需要冷启动下载，先断电→点击下载→上电
4. **安全注意**：3S锂电短路危险，测试时做好绝缘保护
5. **电机匹配**：确认电机KV值与电调PWM频率匹配
